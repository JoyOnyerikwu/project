This is a tool that will be useful for managing
complex and uncertain projects. As seen previous to this page, this
site will require a login for members. The curent page will allow the
user to select and upload a file that contains all necessary information
about any project that a member wants more information about. For example:
The average length that it will take to complete the project shown by a gantt chart
that displays the most efficient process. Once the file is uploaded the user
can then calculate these results. The user will then be
brought to a page where the resulting gantt chart is shown.
